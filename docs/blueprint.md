# **App Name**: Inspire MANIT Official Website

## Core Features:

- Dynamic Image Gallery: Fetch and display images from the backend in a Masonry layout.
- Member Directory: Fetch and display members' information, including roles and responsibilities.
- Alumni Showcase: Display alumni profiles with details on their achievements and contributions.
- Upcoming Events Section: Show upcoming events with registration links and expandable descriptions.
- Contact Form: Allow users to send messages via a functional form, integrated with Web3Forms.
- Dynamic Database Seeding: Automatically populate the database using initData.js.
- Radial Glow Generation: The AI app builder tool will use reasoning to apply the correct radial gradient glow effect on section backgrounds to ensure a smooth design across different components.

## Style Guidelines:

- Background color: Dark background using #0b0f1a to provide a modern feel. Ensure all text is easily readable against the dark backdrop.
- Primary color: Blue accent using #3b82f6. The selected blue offers versatility when combined with other colors, and is distinct from the dark background.
- Accent color: Cyan accent using #00bcd4. The selected cyan differs markedly from the primary blue in both brightness and saturation to establish an appropriate degree of visual contrast.
- Body and headline font: 'Inter' (sans-serif) for a modern, machined, neutral aesthetic across all content.
- Implement a Masonry layout for the image gallery using MUI ImageList, optimizing responsiveness and visual appeal.
- Add a scale-up logo animation in the Hero section to create an engaging visual experience upon page load.
- Utilize MUI Grid settings to prevent overlapping elements on smaller screens (Core Team, Alumni), ensuring a consistent and responsive layout across devices.
